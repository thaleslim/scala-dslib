LinkedList 

class LinkedList[T] extends List[T] with Aggregate[LinkedListIterable[T]]

A trait List' implementation using a group of nodes.

This' nodes are capable of pointing to the next index, which together represent a sequence. The trait List' linearity is given by this behaviour instead of the node' physical placement.
